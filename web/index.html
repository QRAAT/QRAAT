QRAAT - Quail Ridge Automated Animal Tracking

This is the future site of QRAAT, an automated animal tracking system that will provide location data for very small VHF radio tags within the Quail Ridge Reserve, located on Lake Berryessa in Napa county, California.

Learn more about The Quail Ridge Reserve.
